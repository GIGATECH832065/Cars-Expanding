# GigaTech Expanding Cards Project

This project is a simple expanding cards effect showcasing high-performance cars. It includes a flexbox-based layout with image cards that expand upon hover. The cards display information about each car and are styled with a neon blue footer that includes copyright information.

## Project Overview

The **GigaTech Expanding Cards Project** is a web-based interactive card layout. It allows users to hover over various car images, each card expanding to reveal more details about the vehicle. The page also includes a fixed neon blue footer with the copyright text.

## Features

- Expanding cards with hover effect.
- Interactive layout with images of high-performance cars.
- Neon blue footer with the copyright text.
- Fully responsive design, optimized for both desktop and mobile.

Technologies Used
HTML: Structure of the webpage and content.
CSS: Styles for layout, design, and responsive behavior.
JavaScript: Logic for the interactive expanding cards effect.
Usage
HTML: The structure of the project is contained in the index.html file. The body contains the car panels, and the footer includes the copyright notice.

CSS: The styles are located in style.css. The footer is styled with a neon blue color and a glowing effect. The cards expand and contract on hover with a smooth animation effect.

JavaScript: The script file (script.js) handles the logic for expanding the cards. It listens for user interactions (hovering over the panels) and adjusts the size and content display accordingly.
